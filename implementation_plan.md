# 台灣政治模擬器 (Taiwan Political Simulation) - 遊戲設計草案

目標是打造一款以台灣總統大選為背景的策略模擬遊戲，結合地區政治版圖、議題攻防與資源管理，體驗選戰的刺激感。

> [!IMPORTANT]
> **需要您的審查與決定**
> 請檢閱以下的遊戲核心機制設計，確保符合您心中的願景。此外，我們需要決定**開發技術的選擇**，才能開始撰寫程式碼。

## 1. 核心機制與勝利條件

不同於美國大選的「選舉人團制」，台灣總統選舉採「**普選制 (Popular Vote)**」，得票數最高者當選。
因此，遊戲的核心策略不再是爭取特定州，而是**「極大化總得票數」**。人口密集的「六都」（新北市、台中市、高雄市等地）將成為兵家必爭之地，而其他縣市的選票也可能成為膠著時的關鍵。

*   **遊戲節奏**：採用回合制。每 1 回合代表選戰中的「1 週」，總共進行 N 回合（例如 20 回合，象徵選前半年）。
*   **勝利條件**：遊戲結束（第 N 回合後結算），獲得全台灣最高總票數的候選人獲勝。

## 2. 遊戲地圖與選票分佈

*   **地圖劃分**：台灣 22 個縣市（含離島），每個縣市視為一個可互動的節點。
*   **人口比例**：每個縣市有對應的「選舉人數」。例如新北市可能有三百多萬票，而連江縣只有一萬多票。
*   **基本盤（政治版圖）**：每個縣市有初始的藍/綠/白傾向。例如南部某些縣市起初偏向某陣營，北部或客家庄偏向另一陣營。

## 3. 資源與屬性系統

玩家在每回合需要分配有限的資源來最大化影響力。

### 候選人資源（每回合重置或累積）：
*   ⚡ **體力 (Stamina / AP)**：每回合的行動點數。移動到其他縣市、發表演說、舉辦造勢大會都需要消耗體力。
*   💰 **資金 (Funds)**：用於購買電視/網路廣告、建立地方競選總部或聘請網軍。
*   ⭐ **政治資本 (Political Capital)**：用於發動特殊卡牌（例如：拔樁、抹黑對手、請大咖站台）。

### 候選人屬性（靜態能力）：
*   **魅力 (Charisma)**：影響演說時的群眾渲染力以及募款速度。
*   **募款能力 (Fundraising)**：每回合固定帶來的基本資金收入。
*   **在地人脈 (Local Roots)**：於特定區域（如中南部）的初始好感度加成。

## 4. 行動清單 (Actions)

玩家在每個回合可以執行的動作（受體力與資金限制）：
1.  **移動 (Travel)**：搭乘高鐵/飛機前往目標縣市，不同距離可能消耗不同體力。
2.  **發表演說 (Speech)**：在當前縣市針對「特定議題」發表演說，提升該議題支持者的好感度。
3.  **舉辦造勢晚會 (Rally)**：大量消耗資金與體力，一次性大幅提升當前縣市的支持度，並激發媒體聲量。
4.  **建立競選總部 (Build HQ)**：在該縣市花費資金建立總部，每回合自動提供該縣市微幅支持度成長與在地情報。
5.  **募款餐會 (Fundraise Event)**：消耗體力換取大量資金，可能依賴當地基本盤。
6.  **投放廣告 (Run Ads)**：不需移動，直接花費資金在特定縣市（或全國）投放針對對手的負面廣告或自身形象廣告。

## 5. 核心議題系統 (Issues)

選民不是均質的，各縣市對不同議題的重視程度不同。
*   **預設議題**：兩岸關係、經濟與房價、能源政策（核能/綠電）、社會福利、少子化。
*   例如：在「新竹市」演說，主打「經濟與科技發展」效果極佳；在「中南部」主打「基礎建設」可能更受歡迎。
